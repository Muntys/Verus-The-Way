---

## 🆕 Good First Issue: [RUST-01] Verified VerusID Status Check

**Difficulty:** Beginner (Formal Verification) / Intermediate (Rust)
**Status:** Open 🟢

### The Challenge
Create a formally verified Rust function that checks if a **VerusID** is currently "Locked" or "Unlocked" by querying the protocol via VerusIDX.

### Technical Requirements:
1. **The Spec:** Define a specification function `is_locked_spec` that represents the protocol-level "Vault" state.
2. **The Logic:** Implement a Rust function that parses the `getidentity` JSON response.
3. **The Proof:** Provide an SMT-based proof that the function correctly identifies a `flags: 2` (Locked) state and prevents any "Spend" actions while locked.

### Getting Started:
1. Install [Verus-Lang](https://github.com).
2. Use the **VerusIDX `identity` server** to fetch a test ID.
3. Submit your PR with the `.rs` file containing both code and proofs.
